How Do I Change the Dates?

To change the dates, open the settings.py file using notepad or a different editor. You'll see something like the following lines:

 

            start_date = [2018, 9, 28]  # YYYY, MM, DD
            end_date = [2018, 12, 14]  # YYYY, MM, DD
Go ahead and change them to whichever dates you'd like to start and end at. For example, if you'd like the starting jumu'ah to be January 4th, 2019, change the value of start_date to be [2019, 1, 4]. Note: changing the dates will automatically delete all locally-saved data! There is no way to recover this data.

How Do I Change the Email Templates?

To change the templates, open the settings.py file using notepad or a different editor. You'll see something like the following lines:

 

            khateeb_request_email = ...
            khateeb_reminder_email = ...
Go ahead and change the templates to whatever you like. You can use HTML tags in the templates.
